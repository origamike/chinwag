Instructions

  1. Once for each page with dialogue,
    copy what's in the box and paste it into one of these places

    • your .css file
    • between <style> and </style> tags in your .html file's <header>

  2. Each time you want to add dialogue to that page,
    use HTML elements with either the p1 or p2 class:

    When Person 1 talks, make the code look like this: <p class="p1">*whatever Person 1 says*</p>

    And when Person 2 talks, use <p class="p2">*whatever Person 2 says*</p>

    Feel free to copy and paste those snippets from < to > and replace everything from * to * with your words, if that's easier than adding classes to HTML elements.

  3. Double-check that all of your brackets, quotation marks, and slashes are in the right places.

  4. Enjoy some freshly skewed chinwags!

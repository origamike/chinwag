Chinwag

Chinwag does
this
to dialogue:

Have fun stormin' da castle.

Think it'll work?

It'd take a miracle.

Here's how to use it:

  1. Once for each page with dialogue, EITHER...
    copy the bracketed text from the next line and paste it inside your page's .html file just before </head>:
    <link rel="stylesheet" href="http://origamike.github.io/stylesheets/chinwag.css">

    OR...
    copy what's in this little box under here and paste it into one of these places:

    • your linked .css file
    • between <style> and </style> tags in your .html file's <head>, if you see all of those tags there

  2. Each time you want to add dialogue to a page,
    type it into HTML elements with either the p1 or p2 class:

    When Person 1 talks, make the code look like this: <p class="p1">*whatever Person 1 says*</p>

    And when Person 2 talks, use this code: <p class="p2">*whatever Person 2 says*</p>

  3. Double-check that all of your brackets, quotation marks, and slashes are in the right places.

  4. Enjoy some freshly skewed chinwags!
    Email or tweet if you have questions or a page to show off.
